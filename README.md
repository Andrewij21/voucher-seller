# Muatparts Seller

Dokumentasi singkat Muatparts

## Installation

```bash
npm install
```

## Usage

```
npm run dev
```

## Routes

Semua route web page bisa dilihat berdasarkan nama folder
contoh :

- folder buatVoucher -> http://localhost:3000/buatVoucher

### Note

Api AZL belum di integrasikan ke dalam web apps
